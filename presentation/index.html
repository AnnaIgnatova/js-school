What is Electron?

Electron.js is the most popular framework for desktop app development, that allows the user to create applications with HTML5, CSS, and JavaScript

A little bit of history of Electron.js

Key features of Electron.js

Electron JS uses web technologies like simple HTML, CSS, and JavaScript. It does not require native skills unless you want to do something advanced. It can be designed for a single browser. Its file system belongs to Node.js APIs and works on Linux, Mac OS X, Windows.

what are the advantages of electron?

Cross-platform support

Low barrier to entry

Increased development speed

Automatic updates and convenient installers

Large community of developers and users

How to create apps with Electron

create the package.json file

install electron.js


          npm install electron --save-dev

create the index.js file

create the index.html file


            <!DOCTYPE html>
            <html>
            <head>
              <meta charset="UTF-8">
              <title>Electron App</title>
            </head>
            <body>
              <h1>Hello World!</h1>
            </body>
            </html>

build electron app


          npm start
